# Ultimate Typing Speed Test

A Java Swing typing speed test app with difficulty levels, live stats, and a leaderboard.

## Requirements

- Java JDK 8+ installed

## Run

From the project root:

```bash
java -cp . ultimatetypingspeedtest.UltimateTypingSpeedTest
```

If you need to recompile:

```bash
javac -d . UltimateTypingSpeedTest.java
```

## Notes

- Results may be saved to TypingSpeedTestResults.txt.
- Compiled .class files can be deleted and regenerated with the compile command above.
